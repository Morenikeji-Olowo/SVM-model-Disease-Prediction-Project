About This Project

AI Health Prediction System is an academic project that uses Support Vector Machines (SVM) to predict Diabetes and Breast Cancer. This project demonstrates the use of machine learning for early disease detection and decision support in healthcare.

Project Goals

  • Provide a user-friendly interface to input medical data.
  • Demonstrate SVM-based prediction for Diabetes and Breast Cancer.
  • Showcase frontend and backend integration in a real-world project.
  • Help students understand AI in healthcare applications.
